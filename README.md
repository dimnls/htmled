# htmled

Paste or upload an HTML file, edit any text inline by clicking it, export clean HTML when done.

**[→ Open htmled](https://dimnls.github.io/htmled/)**

## How it works

1. Paste HTML or upload a `.html` file
2. Click any text in the preview to edit it directly
3. Press Enter or Esc to confirm each edit
4. Hit **export** to get clean HTML back — editor scaffolding stripped, structure untouched

## Features

- Paste HTML or drag & drop / upload a `.html` / `.htm` file
- Inline contenteditable editing on all text nodes
- Edit counter tracks changes in the session
- Export: copy to clipboard or download as file
- Zero dependencies, single HTML file
